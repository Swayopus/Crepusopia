Crepusopia Architecture

Astro 5 SSG + Tailwind CSS v4 — Static Blog · Left-to-right data flow

Build Pipeline

  • • pnpm build → astro build + pagefind
  • • MDX + remark (directive, callout, math)
  • • rehype-katex math → HTML+CSS render
  • • Shiki dual-theme syntax highlighting
  • • Tailwind CSS v4 via @tailwindcss/vite
  • • Zero JS by default — fully static HTML
  • • Module aliases (@components, @utils…)

Content & Routing

  • • Content Collections: posts + status
  • • File-based routing: 12 page routes
  • • Pagination: /posts/page/[page]
  • • Tags, categories, archive, friends
  • • Zod schema in content.config.ts
  • • RSS Feed + Sitemap auto-generation
  • • Scaffold: pnpm new-post <filename>

Browser Features

  • • Pagefind full-text search (Ctrl+K)
  • • Dark/Light theme toggle (system pref)
  • • View Transitions API for page nav
  • • TOC: desktop sidebar + mobile FAB
  • • Code copy button + language label
  • • SEO: OG + Twitter Cards + JSON-LD
  • • FOUC prevention + responsive 3-col

Key Integrations

  • • Hitokoto.cn — daily quote API
  • • localStorage cache (per-day expiry)
  • • KaTeX CDN — math (lazy loaded)
  • • Pagefind — static search index
  • • Google Fonts — JetBrains Mono
  • • Social: GitHub · Bilibili · X · Telegram
  • • Typewriter + Hitokoto banner animation
