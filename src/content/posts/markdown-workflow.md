## 核心工具

- **VS Code** — 编辑器，配合 Markdown 插件
- **Prettier** — 统一格式化
- **markdownlint** — 语法规范检查
- **Git** — 版本管理

## 必备 VS Code 插件

1. **Markdown All in One** — 快捷键、TOC、预览
2. **markdownlint** — 规范检查
3. **Code Spell Checker** — 拼写检查
4. **Paste Image** — 快速粘贴图片

## Frontmatter 模板

用脚本自动生成 frontmatter：

```yaml
---
title: ""
description: ""
published: YYYY-MM-DD
tags: []
category: ""
draft: true
---
```

## 小结

好的工具链让写作更专注于内容本身。Markdown + Git 的组合简单而强大。
